How to Install Node.js

Step 1: Visit Official Website

Go to the official Node.js website: https://nodejs.org

Step 2: Download Installer

Click on the LTS (Long Term Support) version for better stability.

Step 3: Run the Installer

Open the downloaded file and follow the installation steps (Next → Next → Finish).

Step 4: Verify Installation

Open Command Prompt or Terminal and run:

node -v
npm -v

If versions appear, installation is successful.

Step 5: Start Using Node.js

You can now run JavaScript using:

node filename.js
